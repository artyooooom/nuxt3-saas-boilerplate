# Nuxt 3 SAAS Landing Builder 
The Nuxt 3 blocks with all you need to build your SaaS or any other web app landing page. Using shadcn-nuxt & tailwindcss.

[See Demo](/)

## Available Blocks
* LandingHeader
* LandingCTA
* LandingFAQ
* LandingHero
* LandingLogos
* LandingPricings
* LandingQuote
* LandingSection
* LandingTestimonials
* LandingFooter

## Template
Explore sample template of a landing page built using our blocks [here](/)

## Setup

```bash
npm install
```

## Development Server
Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
## Production

Build the application for production:

```bash
npm run build
```
