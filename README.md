# Nuxt 3 SAAS Landing Builder 
The Nuxt 3 blocks with all you need to build your SaaS or any other web app landing page. Using shadcn-nuxt & tailwindcss.

[See Demo](/)
[Documentation](/)
[Template]()

---

## Available Blocks
* Header
* LandingHero
* LandingLogos
* LandingCTA
* LandingSection
* LandingQuote
* LandingTestimonials
* LandingPricings
* LandingFAQ
* Footer

## Setup

```bash
npm install
```

## Development Server
Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
## Production

Build the application for production:

```bash
npm run build
```
